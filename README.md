Sistema de **monitoramento comunitário** desenvolvido para centralizar **alertas, notificações e comunicação entre moradores** em tempo real.

O projeto apresenta um **dashboard moderno**, com visualização de incidentes, mapa interativo, estatísticas e notificações organizadas.

---

# 🧭 Visão Geral

O **Comunidade Alerta** é uma plataforma web criada para melhorar a comunicação e segurança em comunidades e bairros.

Com ele é possível:

* acompanhar **alertas em tempo real**
* receber **notificações importantes**
* visualizar incidentes no **mapa**
* monitorar **estatísticas da comunidade**
* organizar alertas por **categoria e prioridade**

# 🚀 Como Executar o Projeto

Clone o repositório:

```
git clone https://github.com/jecziw/comunidade-alerta.git
```

Entre na pasta do projeto:

```
cd comunidade-alerta
```

Abra o arquivo no navegador:

```
dashboard.html
```
